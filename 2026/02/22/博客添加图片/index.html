博客添加图片

Created2026-02-22|Updated2026-02-22
|Post Views:

Butterfly主题修改界面图片核心流程：准备图片→存放至指定目录→修改主题配置文件(_config.butterfly.yml)→本地预览→部署上线。下面按具体场景分步说明，新手也能轻松操作。

一、前置准备与基础规则

1. 图片存放位置

有两种方式管理图片，推荐本地存储更稳定：

存储方式 路径/方法 配置写法示例
本地存储 主题目录：themes/butterfly/source/img/
根目录：source/images/(需手动创建)		 /img/xxx.png 或 /images/xxx.png
网络存储 图床(阿里云OSS、腾讯云COS、SM.MS等) https://xxx.com/xxx.png

注意：本地图片需放在source目录下，Hexo才会将其复制到最终的public目录中。

2. 配置文件位置

修改主题配置文件：_config.butterfly.yml（位于主题目录themes/butterfly/下）。

3. 操作流程

  1. 准备好要替换的图片（建议裁剪合适尺寸，压缩后使用）
  2. 将图片上传到指定目录
  3. 编辑配置文件，修改对应图片路径
  4. 本地预览（hexo clean && hexo s）
  5. 确认无误后部署（hexo clean && hexo g && hexo d）

二、常见界面图片修改方法

1. 网站图标（Favicon）

  • 配置位置：
  • 操作步骤：
    1. 将图标文件（建议png格式，尺寸16×16或32×32）命名为favicon.png
    2. 上传到themes/butterfly/source/img/目录
    3. 修改配置文件中的路径为你的图标文件名

2. 主页封面图（Banner）

  • 配置位置：
  • 操作步骤：
    1. 准备封面图（建议尺寸1920×1080，比例16:9）
    2. 上传到themes/butterfly/source/img/
    3. 修改index_img的值为图片路径

3. 网站背景图

  • 配置位置：
  • 操作步骤：
    1. 准备背景图（建议高清、浅色，避免影响文字阅读）
    2. 上传到指定目录
    3. 修改background.image的值为图片路径

4. 头像（个人资料图片）

  • 配置位置：
  • 操作步骤：
    1. 准备头像图片（建议正方形，尺寸200×200）
    2. 上传到themes/butterfly/source/img/
    3. 修改avatar.img的值为头像路径

三、高级技巧与常见问题

1. 图片路径问题排查

如果图片不显示，按以下步骤排查：

  1. 确认图片路径是否正确（区分相对路径和绝对路径）
  2. 执行hexo clean清理缓存后重新生成
  3. 检查图片文件名是否有空格或特殊字符（建议使用英文和数字）
  4. 本地预览时，通过浏览器开发者工具（F12）查看网络请求，确认图片是否404

2. 图片优化建议

  • 压缩图片：使用TinyPNG等工具压缩，减少加载时间
  • 合理尺寸：不同位置使用合适尺寸，避免大图片小显示
  • 格式选择：图标用PNG，背景用JPG，透明背景用PNG

四、总结与下一步

  1. 修改Butterfly主题图片的核心是修改_config.butterfly.yml配置文件+放置图片到正确目录
  2. 先本地预览确认效果，再部署到线上
  3. 建议将所有自定义图片统一管理，方便后续维护

按照以上步骤，你可以轻松修改Butterfly主题中所有界面图片，打造个性化的博客外观。如果需要修改其他图片（如社交图标、按钮图标等），可在主题配置文件中搜索相关关键词（如social、icon）找到对应配置项。
